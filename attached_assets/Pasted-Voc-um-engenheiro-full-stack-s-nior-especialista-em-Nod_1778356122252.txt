# ❌ PROBLEMAS ATUAIS

## 1. ERRO DE ROTA (MAIS CRÍTICO)
- Usuário recebe "rota não encontrada" ao acessar o site
- React Router não funciona corretamente no deploy
- Refresh em qualquer rota quebra a aplicação

## 2. BACKEND NÃO SERVE FRONTEND CORRETAMENTE
- dist/public existe após build
- porém não há fallback para index.html
- rotas do React não são tratadas no servidor

---

## 3. SEED DE USUÁRIOS (INCONSISTÊNCIA POSSÍVEL)
- Usuários admin, gerente, funcionario devem existir automaticamente
- senha deve ser sempre hash com bcrypt
- evitar password_hash NULL
- evitar duplicação de seed no restart

---

## 4. AUTENTICAÇÃO / LOGIN
- login pode falhar se role estiver inválida
- roles válidas: admin, gerente, funcionario
- sessão precisa de SESSION_SECRET configurado
- cookies/sessão devem funcionar em produção

---

## 5. DEPLOY RENDER
- servidor deve escutar em process.env.PORT
- não pode hardcode de porta
- logs indicam warnings de SSL (normal, não crítico)
